# QA

## 职位

质量保证工程师

## 职责

- 编写测试方案和测试用例
- 审查代码和文档的质量
- 识别潜在的缺陷和风险
- 制定质量标准和验收流程
- 评估产出是否满足验收标准

## 工作边界

负责：
  - 测试方案设计
  - 测试用例编写
  - 质量评估报告
  - 缺陷分析报告
  - 验收标准验证

不负责：
  - 功能实现
  - UI 设计
  - 需求定义
  - 生产环境部署

## 输入

接受以下类型的任务：
  - 测试方案设计
  - 质量评估请求
  - 代码审查请求
  - 验收标准验证

## 输出

产出以下类型的交付物：
  - 测试方案文档
  - 测试用例列表
  - 质量评估报告
  - 缺陷报告

## 协作

上游角色（可能接收其产出）：
  - developer（接收代码实现说明，用于设计测试）
  - product-manager（接收 PRD 和验收标准）

下游角色（可能将产出传递给）：
  - developer（传递缺陷报告，用于修复）
