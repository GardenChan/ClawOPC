# Edit Content

## 触发条件

当任务 instruction 要求编辑或润色已有内容时，使用此 skill。

## 执行步骤

### 步骤 1：理解编辑需求

读取 task.md 中的：
  - 需要编辑的原始内容
  - 编辑方向（润色/缩短/改变语气/重构等）
  - 目标受众和使用场景

### 步骤 2：分析原文

评估原文的：
  - 结构是否合理
  - 语言是否流畅
  - 信息是否完整
  - 风格是否合适

### 步骤 3：执行编辑

按照编辑方向修改内容，
保留原文的核心信息，
改善表达方式和结构。

## 产出格式

```
## 产出摘要

[编辑概述：做了哪些修改]

## 产出内容

[编辑后的完整内容]

## 说明与备注

[主要修改点说明、修改理由]
```
