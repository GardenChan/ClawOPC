# Write Copy

## 触发条件

当任务 instruction 要求撰写文案或内容时，使用此 skill。

## 执行步骤

### 步骤 1：理解需求

读取 task.md 中的：
  - 内容类型（产品介绍/博客/营销文案/帮助文档等）
  - 目标读者
  - 核心信息和关键卖点
  - 语气和风格要求
  - 长度要求

### 步骤 2：制定大纲

根据内容类型制定结构：
  - 标题（吸引注意力）
  - 开头（建立共鸣或提出问题）
  - 正文（核心内容，逻辑清晰）
  - 结尾（Call to Action 或总结）

### 步骤 3：撰写内容

写作要求：
  - 语言流畅，不生硬
  - 段落简短，易于阅读
  - 关键信息突出
  - 符合目标受众的阅读习惯

### 步骤 4：自审

检查：
  - 是否回答了读者关心的问题
  - 是否有多余的句子可以删除
  - 语气是否一致
  - 有没有错别字和语病

## 产出格式

```
## 产出摘要

[内容概述：什么类型、面向谁、核心信息]

## 产出内容

[完整文案内容]

## 说明与备注

[写作思路、目标受众分析、替代方案]

## 验收自查

- [x] 核心信息完整传达
- [x] 语气符合要求
- [x] 结构清晰易读
- [x] 无错别字和语病
```
